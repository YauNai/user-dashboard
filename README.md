# 虛構讀書社交平台「悅讀」會員頁面

此專案專注於網站前端的後台系統 UI / UX，包含資料統計、會員資料修改、文章編輯發布等功能。

[線上預覽](https://user-dashboard-1p36.onrender.com)

## 網站特色

- 響應式設計
- 完全使用 DaysiUI 和 TailwindCSS 提供的 classes
  - 包括但不限於：3D hover card、Modal 等 components
- 圖片上傳與即時預覽

## 使用框架與套件
### 框架

- Vue.js

### 套件

- Vue Router
- Chart.js
- Bootstrap Icons
- DaysiUI
- TailwindCSS

## 網站架構

```
┌─ 總覽
|    ├─ 統計資料
|    └─ 近期活動
├─ 新增紀錄
├─ 個人資訊
├─ 社群
└─ 隱私權與安全性
     ├─ 隱私權設定
     └─ 安全性設定       
```
